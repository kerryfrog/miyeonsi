# 프로젝트 분석: 미연시 시뮬레이터 제작 툴

## 1. 프로젝트 개요

이 프로젝트는 웹 기반의 **미연시 (미소녀 연애 시뮬레이션) 스타일 장면 제작 툴**입니다.

사용자가 직접 테마, 배경, 캐릭터 등을 선택하고 배치하여 게임의 한 장면처럼 꾸민 뒤, 완성된 장면을 이미지 파일로 저장하는 기능을 제공하는 것으로 보입니다.

`app/components/steps` 폴더의 컴포en넌트들 ( `Step1Theme`, `Step2Background`, `Step3Canvas` 등)은 장면 제작이 여러 단계에 걸쳐 진행됨을 시사합니다.

## 2. 주요 기능

- **장면 커스터마이징**: 사용자는 여러 단계에 걸쳐 장면을 구성합니다.
  - 테마 선택
  - 배경 이미지 선택
  - 캐릭터 이미지 추가 및 배치
  - 이미지 배경 제거 (`@imgly/background-removal` 라이브러리 사용)
- **이미지 생성/저장**: `html2canvas-pro` 라이브러리를 사용하여 사용자가 제작한 장면을 캡쳐하여 하나의 이미지로 만들어주는 기능이 핵심입니다.
- **OG 이미지 생성**: `@vercel/og`를 사용하여 소셜 미디어 공유 시 표시될 미리보기 이미지를 동적으로 생성할 수 있습니다.

## 3. 기술 스택

- **프레임워크**: Next.js (v16.1.1)
- **UI 라이브러리**: React (v19.2.3)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **핵심 라이브러리**:
  - `html2canvas-pro`: 웹 페이지의 특정 부분을 캔버스로 렌더링하여 이미지로 변환합니다.
  - `@imgly/background-removal`: 이미지에서 배경을 제거하는 기능을 제공합니다.
- **개발 도구**: ESLint (코드 린팅)

## 4. 프로젝트 구조

- `app/`: Next.js의 앱 라우터 기반으로, 페이지 및 컴포넌트가 위치합니다.
  - `page.tsx`: 애플리케이션의 메인 페이지입니다.
  - `components/`: UI를 구성하는 재사용 가능한 React 컴포넌트들이 있습니다.
    - `steps/`: 장면 제작의 각 단계를 담당하는 컴포넌트들이 모여있습니다.
  - `lib/`: 공통 유틸리티 함수나 상수 값이 정의되어 있습니다.
- `public/`: 이미지, 폰트 등 정적 에셋이 위치합니다.
- `package.json`: 프로젝트의 의존성 및 스크립트가 명시되어 있습니다.
